# Lumen Eats — Premium Restauran

Welcome to the **Lumen Eats** repository! This is a modern, fully responsive multi-page website designed for a premium Spanish and coastal kitchen restaurant. 

## 🎯 Project Goals

The main objectives of this project were to:
* **Master Responsive Web Design:** Build a rock-solid adaptive layout that works flawlessly across all devices, from ultra-compact smartphones (like iPhone SE) to tablets and large desktop monitors.
* **Level Up Frontend Skills:** Deepen my practical knowledge of modern semantic HTML5 and advanced CSS3 features (Flexbox, Grid, advanced positioning, and Media Queries).
* **End-to-End Product Development:** Take a project through the entire lifecycle—from planning the UX/UI concept to generating visual assets and content using modern AI tools.

## 🛠️ Technologies & Tools Used

* **Frontend:** HTML5, CSS3, Basic JavaScript.
* **Design & AI:** Figma, Generative AI tools (for high-fidelity image assets and content creation).

## 📱 Features

* 100% Adaptive and responsive layouts for Mobile, Tablet, and Desktop.
* Premium visual assets and text content created with the help of AI.
* Clean, elegant pixel-perfect typography.
* Beautifully styled interactive layouts and components.
* Structured, maintainable, and readable code.
